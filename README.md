# ARC Raiders Field Guide

Static browser app for helping ARC Raiders players learn the game from beginner fundamentals through advanced raid decision-making.

## Run

Open [index.html](/C:/Users/Jim/OneDrive%20-%20Higgins/Documents/Playground/arc-raiders-guide/index.html) in any modern browser.

## Files

- `index.html`: app shell and responsive layout
- `styles.css`: ARC Raiders-inspired visual system and mobile styling
- `src/main.js`: guide tracks, lesson content, machine intel, and rendering logic
- `site.webmanifest`: install metadata for app-like mobile use
- `assets/arc-signal.svg`: app icon
